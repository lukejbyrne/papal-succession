# Design

## Visual Direction

The site should feel like a papal register laid out in a quiet study: parchment, ink, muted red, brass accents, and restrained historical portraiture.

## Color

- Parchment: `#f5efe1`
- Ink: `#1f1a13`
- Accent red: `#8b1e2d`
- Brass: `#d4a017`
- Muted blue: `#3a4a7a`

## Typography

- Display: Georgia or the existing serif stack.
- Interface/body: system sans.
- Labels: small uppercase with positive tracking.

## Components

- Large literal homepage heading.
- Guided route tiles.
- Horizontal pontificate timeline.
- Mobile-friendly directory rows.
- Pope profile pages with source list and nearby succession.
- Crisis cards for disputed or non-obvious sequence moments.

## Motion

Use only simple hover/focus transitions. The site is a reference tool, so motion should clarify interaction rather than perform.
